## 2.2 Gaussian输入文件结构

### 输入文件特点

- 自由格式，不区分字母的大小写
- 关键字和选项可以使用缩写
- 空格、制表位、逗号和正斜杠（/）为一行内不同项间的分隔符
- 每行的长度小于80个字符

### 输入文件结构

- Link 0 Commands
- Route section(# lines)
- Title section
- Molecule specification
- Optional additional sections

### Link 0 命令行    %Section

用%开始，最常用的是确定chk文件名称及位置，以及内存资源的使用

保留chk文件目的

- 在进一步的计算中直接读取前面的计算结果（分子构型、SCF初始估计值、频率分析等），省时
- 如计算意外终止，可从chk文件保存的位置继续计算

Default.Rou中的设置

- -M- 1200MB
- -#- MaxDisk=2GB
- -p- 4

### 路径部分（Route Section）

以“#”开始，定义理论方法、基组和计算的类型

- eg：#T RHF/6-31G(d) SCF=Tight Opt

### 标题行（Title Section）

对文件进行说明，不能是空行

### 分子说明部分（Molecule Specification）

电荷和自旋多重度（spin multiplication）

自旋多重度=2S+1
